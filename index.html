História da Web

       A idéia da Compução é bem antiga, no entando o primeiro computador criado com a intenção de fazer cauculos mais complexos foi o Mark I (1944). Eram computadores gigantescos que funcionavam atraves de vauvulas eletromagnéticas e ocupavam salas inteiras de um predio.

Logo após em 1946 foi lançado o Colossus pelos alemães e Eniac pelos americanos, ambos foram muinto utilisados para fazer cauculos de guerra

      A segunda geração de computadores foi marcada pela "reduçao" no tamanho dos computadors, qundo na primeira geração era necessário varias salas para colocar um computador, na segunda geração já era possivel colocar em uma unica sala varios computadores. Tudo isso graças a substituição das vaulvulas eletromagneticas por transistors.

      Já na terceira (1965 a 1970) geração foi a ves dos circuitos integrados que teve como consequencia a diminuição ainda mais dos computadores.

      A quarta geração consiste apartir de ( 1971 até os dias atuais) foi passado a ser utilizados microprocessadores, ou seja, apartir dai foi surgindos computadores mais portateis e mais potentes.

O Que são Clients

Cliente no vocabulario de programação, são basicamente dispositivos como computadores, celulares, notebooks, que se comunicam com o servidor atraves da internet, ou seja, consomem informações. Esse acesso em muitos casos são feitos pelos navegadores que interpretam o html, css e javascript.

Esses navegadores seguem o padraão W3C que é orgão responsavel por padronizar as tecnologias web, como o html e o css. Embora todos os principais navegadores seguirem o padrão w3c, cada um tem suas proprias particularidades.

Aplicações web

Aplicações web são programas que você usa diretamente no seu navegador de internet, como um site. Não é necessário instalá-los, eles funcionam em diferentes dispositivos e sempre estão atualizados. Alguns exemplos são o Gmail e o Facebook. Eles se comunicam com servidores remotos para funcionar e são essenciais para muitas coisas online.

Sobre servidores

Servidores são como computadores especiais que guardam informações e as compartilham com outros dispositivos quando necessário. Eles são importantes para a internet e outros sistemas de rede. Por exemplo, quando você acessa um site, o servidor do site envia a página para o seu computador. Eles fazem isso com várias coisas, como sites, e-mails e arquivos. Servidores são como computadores especiais que armazenam informações e as compartilham com outros dispositivos quando necessário. Existem diferentes tipos de servidores, como servidores web (para sites), servidores de e-mail (para e-mails), servidores de arquivos (para armazenamento) e servidores de banco de dados (para armazenar dados organizados). Eles são vitais para a internet e redes.

Hardware e Software para servidores

Hardware em servidores são as parte fisacas do computador, assim como nos computadores pessoais, já os software são a parte logíca dos computradores, no caso dos computadores servidores são sistemas opereacionais expecíficos para estes fins, como Windows Server, CentOS, debian, fedora ubuntu Oracle. Alem disso existe tambme os softwere especificos para servir, como o APACHE e o NGINX.
